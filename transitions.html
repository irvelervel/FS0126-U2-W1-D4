Le TRANSIZIONI in CSS

Una TRANSIZIONE rappresenta un'applicazione di una o più regole CSS in maniera graduale, "rallentata". Una TRANSIZIONE viene applicata a seguito di un cambio di stato in CSS, quello più comune è il cambio di stato derivato da un evento di HOVER (quando il cursore del mouse si sovrappone ad un elemento HTML).
